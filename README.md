### 작성 규칙
- 문서 확장자는 `.md`를 사용한다.
- 언어나 기술명으로 폴더를 만들고, 그 아래에 새로운 문서를 생성한다.
- Commit 할 때는 `[말머리]` 를 사용한다.
- `Blog`와 `repo` 모두 새롭게 알게된 것들 위주로 작성한다.
