### 8진수와 16진수

---

🗒️ Java와 리터럴은 8진수와 16진수를 지원한다
<br/><br/>

- **8진수** (`Octal` → 0 to 7)
    
    ```java
    > int eight = 010;
    eight ==> 8
    ```
    
    `0`으로 시작하면 8진수

<br/>

- **16진수** (`hexadecimal` → 0 to 9, A, B, C, D, E and F)
    
    ```java
    > int sixteen = 0x10;
    sixteen ==> 16
    ```
    
    ```java
    > int fifteen = 0xF;
    fifteen ==> 15
    ```
    
    `0x`로 시작하면 16진수
