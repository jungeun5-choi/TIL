### `Math.-()`

* 수학에서 자주 사용하는 상수들과 함수들을 미리 구현해 놓은 클래스

    📝 보충 자료: [http://www.tcpschool.com/java/java_api_math]


---

- **`Math.random()`** : `double` type으로, 0.0이상 1.0 미만 사이의 값을 반환
    
    ```java
    > Math.random()
    $9 ==> 0.9724216879327302
    ```
    
- **`Math.min(숫자1, 숫자2)`** : 숫자 1과 숫자 2를 비교하여 더 작은 숫자를 반환
    
    ```java
    > Math.min(77,3)
    $10 ==> 3
    ```
